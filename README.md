# Portfolio	

개인 프로젝트 호텔사이트
## 기술 스택
- 개발기간 :  2020/09 ~ 2020/12
- 언어
  - Javascript / node.js
  
 - 프레임워크 
   - React
   - redux (react-redux)
   - styled-components
   
- 클라우드
   - AWS
   
- DATABASE 
  - mongoDB / mongoose
 
 
 # 주요기능
  AWS 무료 라이센스 사용하여 운영체제 Ubuntu에서 서버를 pm2로 작동하였습니다.
  
  사용자가 호텔 사이트내에서 예약 및 예약 확인이 가능합니다.
