 # 주요기능
  AWS 무료 라이센스 사용하여 운영체제 Ubuntu에서 서버를 pm2로 작동하였습니다.
  
  사용자가 사이트내에서 회원가입,로그인 가능합니다.
  bcrypt모듈을 사용하여 회원가입시 해쉬함수를 더해 비밀번호를 암호화해서 데이터베이스에 저장했습니다.  
  데이터 요청시 jsonwebtoken모듈을 사용하여 인증을구현했습니다.
